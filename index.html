Finals Lab Task 1 - MySQL Basics (Multi-Level Company)

This lab activity focuses on building a database for a company with multiple levels using MySQL. I created tables, inserted data, and executed SQL queries to retrieve required information.

Steps I Followed

  • Created a new database:
    • Used CREATE DATABASE to make the database
    • Used USE to select the database
  • Created tables:
    • Used CREATE TABLE for Employees, Departments, etc.
    • Assigned correct data types (e.g., INT, VARCHAR, DATE)
    • Set PRIMARY KEY and FOREIGN KEY for relationships
  • Checked table structures:
    • Used DESCRIBE or phpMyAdmin to review the design
