Finals Lab Task 1 - Database Normalization

This page contains my lab task on normalizing database tables to reduce redundancy and improve data integrity.

Normalization Process

  • Identified repeating groups and removed them
  • Created separate tables for related data
  • Defined primary and foreign keys
  • Ensured data dependencies are on keys
  • Achieved 1st, 2nd, and 3rd Normal Forms

Results

  • Reduced data redundancy
  • Improved data consistency
  • Optimized database structure
